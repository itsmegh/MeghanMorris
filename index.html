Conscious Cuisine

A team project for University of Denver's Coding Bootcamp.

The first team project gave us the opportunity to integrate information from APIs into a web application. We chose to create a nutrition based application allowed users to search for an ingredient and retrieve recipe ideas with nutrition information included for each result. We used the Yummly and Edamam APIs. The included version is my redesign using Bootstrap.

Train Schedule with Firebase

Coding Bootcamp Assignment

In this assignment, I was tasked with creating a train schedule application that incorporates Firebase to host arrival and departure data. My app retrieves and manipulates this information with Moment.js. My application provides up-to-date information about various trains, namely their arrival times and how many minutes remain until they arrive at their station.

  • Date: November 2018
  • Live Application
  • Github Code
  • Category: Web Development

GifTastic
